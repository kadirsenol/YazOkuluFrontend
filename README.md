# Frontend Proje (.Net Core MVC(Razor View)) - README

## Ön Koşullar
- .NET 8.0 SDK
- Visual Studio 2022 veya üstü (VS Code için C# eklentisi)

---

## 1. Visual Studio ile Çalıştırma

1. Projeyi Visual Studio’da açın.
2. `F5` veya **Start Debugging** ile çalıştırın.

---

## 2. VS Code ile Çalıştırma

1. VS Code ile proje dizinini açın.
2. Terminal açın ve `YazOkuluFrontend` projesinin path’ine gidin:
3. ...\YazOkuluFrontend\YazOkuluFrontend>
4. dotnet run
